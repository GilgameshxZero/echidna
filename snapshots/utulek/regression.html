Regression

May 30, 2023

We use the term regression as opposed to classification when predicting quantitative outputs.

Linear regression is particularly special as it results as the same solution via four methods of estimating the same problem: to best fit a linear line to some set of data. Notably, because the least squares and maximum likelihood methods both give the same regression coefficients, this means that the (multi-variate) regression “distribution” has both mean and mode at the given coefficients.

Projection

It is by definition that regression establishes the projection of YYY onto the column space of XXX. As we are manipulating the columns of XXX, the regression coefficients BBB must be right-multiplied:

Y^=XB.\hat Y=XB. Y^=XB.

Of course, BBB is a 222-by-111 matrix. More generally, projections minimize the norm of the error term EEE, a nnn-by-111 matrix, necessarily orthogonal to XXX’s column space:

Y=XB+E.Y=XB+E. Y=XB+E.

The norm of the error term, ∣E∣|E|∣E∣, is precisely the residual sum-of-squares (RSS):

RSS(Y,Y^)=∣E∣=∑i=1n(y^i−yi)2=(XB−Y)T(XB−Y).RSS(Y,\hat Y)=|E|=\sum_{i=1}^n (\hat y_i - y_i)^2=(XB-Y)^T(XB-Y). RSS(Y,Y^)=∣E∣=i=1∑n​(y^​i​−yi​)2=(XB−Y)T(XB−Y).

Briefly, we review the projection solution. Recall for AAA whose column space draws a line, onto which we wish to project b\mathbf{b}b:

p=Ax^=Pb.\mathbf{p}=A\hat x=P\mathbf{b}. p=Ax^=Pb.

Changing variables into the regression problem, we have

Y^=XB=PY.\hat Y=XB=PY. Y^=XB=PY.

For the line projection, we have by the definition of dot products

x^=ATbATA=(ATA)−1ATb\hat x=\frac{A^T\mathbf{b}}{A^TA}=(A^TA)^{-1}A^T\mathbf{b} x^=ATAATb​=(ATA)−1ATb

and thus in the regression problem we have

B=(XTX)−1XTYB=(X^TX)^{-1}X^TY B=(XTX)−1XTY

as desired.
